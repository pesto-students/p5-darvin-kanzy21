# Redux Steps App


Increment Step and Reset Step counts


## Tech

- Reactjs
- BootStrap
- Redux

## Installation

This App requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd redux-exercise-2
npm i
npm run dev
```

For production environments...

```sh
npm run build
```
